# i_OrderforUs
This is a recording app for keeping tracking of takeaway order and notifying everyone with one-click email function.

Bulid with fully native Swift and MVC.

# Technologies used:

UIKit <i class="fa-solid fa-toolbox">

UserDefaults

MessageUI
